This project is a simple and stylish web form designed using HTML and CSS. It collects basic contact information such as name, email, age, gender, and language preferences. The form also includes modern HTML5 input types like date, time, URL, and file upload.

✨ Features

 Custom-styled file upload button (removes “No file chosen” text)

 Input fields for personal and contact details

 Supports date, time, and local datetime inputs

 Language selection with checkboxes

 Designed with a pink gradient background and smooth rounded corners

 CSS styling for layout, spacing, and typography

 Technologies Used

HTML5 – For the structure and form elements

CSS3 – For styling, layout, and responsive design
